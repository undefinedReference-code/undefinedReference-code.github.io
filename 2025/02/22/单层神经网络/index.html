吴恩达学习笔记 —— 单层神经网络

单层神经网络

以一个二分类模型为例子，模型描述如下：

  • 给出待判断的物体的n个特征x1,x2,...xnx_1,x_2, ... x_nx1​,x2​,...xn​
  • 给出待判断物体的类别：0或1

模型建立：令n个w, 记为：w1,w2,...,wnw_1,w_2,...,w_nw1​,w2​,...,wn​，和常数b，令其满足如下公式：

P=σ(w1∗x1+w2∗x2+...+wn∗xn+b)P=\sigma(w_1*x_1+w_2*x_2+...+w_n*x_n+b) P=σ(w1​∗x1​+w2​∗x2​+...+wn​∗xn​+b)

其中：σ(x)=11+e−(x)\sigma(x)={1\over1+e^{-(x)}}σ(x)=1+e−(x)1​.当P大于0.5时，认为其属于类1，当P小于0.5时，认为其属于类0.

向量化地表示上面的公式，令：

W=(w1,w2,...,wn)W=(w_1,w_2,...,w_n) W=(w1​,w2​,...,wn​)

X=(x1,x2,...xn)TX=(x_1,x_2, ... x_n)^T X=(x1​,x2​,...xn​)T

则X是个n*1矩阵，而W是个1*n的矩阵，那么模型可以简单写成：

Z=W∗X+bZ=W*X+b Z=W∗X+b

A=σ(Z)A=\sigma(Z) A=σ(Z)

上面的乘法是矩阵意义上的乘法，A表示预测的结果，可以理解为有A的概率属于某一类。

机器学习

所谓学习，就是通过给定一系列训练集（即已知的m个{X(m),y(m)X^{(m)},y^{(m)}X(m),y(m)} 标签对），训练出合适的参数W和b，使得模型对未知物体的预测性能良好。

训练模型

训练模型的思想是：对于每一个训练集中的X(m)X^{(m)}X(m)，利用当前的参数W和b计算出a(m)a^{(m)}a(m), 如果和训练集中已知的y(m)y^{(m)}y(m)基本吻合，那么认为模型已经训练的差不多了，如果相差很大，那么需要调整W和b的值，使得两者相差变小。所以下面有几个技术问题：

如何判断预测值和真实值的差距

为了衡量单个样本的训练情况，我们定义了损失函数，lost function：

L(a(i),y(i))=−y(i)log⁡(a(i))−(1−y(i))log⁡(1−a(i))\mathcal{L}(a^{(i)}, y^{(i)}) = - y^{(i)} \log(a^{(i)}) - (1-y^{(i)} ) \log(1-a^{(i)}) L(a(i),y(i))=−y(i)log(a(i))−(1−y(i))log(1−a(i))

当L(a(i),y(i))\mathcal{L}(a^{(i)}, y^{(i)})L(a(i),y(i))取极小值时，模型的训练效果最好，极限情况下，如果完全没有错误，我们人为定义 0∗log0=00*log0 = 00∗log0=0,因此损失函数为0代表完全没有错误，显然这是几乎不可能达到。

损失函数只是衡量了对于一个样本的预测情况，如果有m个样本，很显然为了衡量整体的预测情况，我们引入代价函数，cost function：

J=1m∑i=1mL(a(i),y(i))J = \frac{1}{m} \sum_{i=1}^m \mathcal{L}(a^{(i)}, y^{(i)}) J=m1​i=1∑m​L(a(i),y(i))

代价函数定义为损失函数的平均值，很显然，不同损失函数之间不会相互关联，因此降低每个损失函数的值，有助于降低整体的代价函数J的值

如何优化W,b的值

当损失函数J很大的时候，我们需要一个算法，调整W和b的值，希望能够降低J的值，因此，引入下面的梯度下降法：

首先对于单个训练样本的情况（此时J=L), a(i)a^{(i)}a(i)只有一个，直接记为a，我们有以下的公式：

wi=wi−λdJdwiw_i =w_i - \lambda {dJ\over{dw_i}} wi​=wi​−λdwi​dJ​

b=b−λdJdbb =b - \lambda {dJ\over{db}} b=b−λdbdJ​

其中λ\lambdaλ是一个认为规定的参数，称为学习率。采用链式法则进行求导可以得到：

∂J∂a=a−ya(1−a)\frac{\partial J}{\partial a} = \frac{a-y}{a(1-a)} ∂a∂J​=a(1−a)a−y​

dadz=a∗(1−a){da\over dz}={a*(1-a)} dzda​=a∗(1−a)

∂z∂wi=xi{\partial z\over \partial w_i}={x{_i}} ∂wi​∂z​=xi​

∂z∂b=1{\partial z\over \partial b}={1} ∂b∂z​=1

基于上面公式可以得到：

∂J∂wi=xi∗(a−y){\partial J\over{\partial w_i}}={x_i*(a-y)} ∂wi​∂J​=xi​∗(a−y)

∂J∂b=(a−y){\partial J\over{\partial b}}={(a-y)} ∂b∂J​=(a−y)

但是我们的样本通常不止含有一个训练样本，可以将上面的公式进行修改，以适应m个样本的情况：

∂J∂wi=1m∑j=1mxi(j)∗(a(j)−y(j)){\partial J\over{\partial w_i}}=\frac{1}{m}\sum_{j=1}^{m}{x_i^{(j)}*(a^{(j)}-y^{(j)})} ∂wi​∂J​=m1​j=1∑m​xi(j)​∗(a(j)−y(j))

∂J∂b=1m∑j=1m(a(j)−y(j)){\partial J\over{\partial b}}=\frac{1}{m}\sum_{j=1}^{m}{(a^{(j)}-y^{(j)})} ∂b∂J​=m1​j=1∑m​(a(j)−y(j))

向量化表达

上面所说的主要是使用下面对元素进行表达，下面用更为简洁的线性代数的矩阵表示，进行表达，以达到形式的简洁，同时向量化表达是为了更好地使用编程语言python

设X为一个n行m列的矩阵，每一行代表一个属性，每一列代表一个样本。对应了，每一个样本的标签组成矩阵Y，Y是一个1*m矩阵。

b为常数b为常数 b为常数

X=(X(1),X(2)...,X(m))X = (X^{(1)}, X ^{(2)} ..., X^{(m)}) X=(X(1),X(2)...,X(m))

Y=(y(1),y(2),...,y(m))Y = (y^{(1)},y^{(2)},...,y^{(m)}) Y=(y(1),y(2),...,y(m))

前向传播

Z1∗m=W1∗n∗Xn∗m+b1∗1Z_{1*m} = W_{1*n}*X_{n*m}+b_{1*1} Z1∗m​=W1∗n​∗Xn∗m​+b1∗1​

A=σ(Z)A = \sigma(Z) A=σ(Z)

这里为什么可以利用一个数去加一个矩阵，这是利用了python numpy库的特点，其会将b复制m份形成（1，m）的向量。

后向传播

∂J∂W1∗n=1m(A−Y)1∗mXm∗nT{\partial J\over{\partial W}}_{1*n}=\frac{1}{m} (A-Y)_{1*m}X^{T}_{m*n} ∂W∂J​1∗n​=m1​(A−Y)1∗m​Xm∗nT​

∂J∂b=1m∑j=1m(a(j)−y(j)){\partial J\over{\partial b}}=\frac{1}{m}\sum_{j=1}^{m}{(a^{(j)}-y^{(j)})} ∂b∂J​=m1​j=1∑m​(a(j)−y(j))

将上面两式的结果记为 dW和db，则：

W1∗m=W1∗m−λdW1∗mW_{1*m} = W_{1*m}-\lambda dW_{1*m} W1∗m​=W1∗m​−λdW1∗m​

b=b−λ∗dbb = b-\lambda*db b=b−λ∗db

总结

通过循环进行前向传播和后向传播，不断优化参数，提高模型的泛化力。
